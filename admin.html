📋 Instruções de Upload

Como fazer upload no Cloudinary:

  1. Acesse: cloudinary.com/console
  2. Faça login com: Cloud Name: dygbrcrr6
  3. Clique em "Media Library" → "Upload"
  4. Selecione seu arquivo MP3
  5. Após o upload, clique no arquivo
  6. Copie a URL pública (terminada em .mp3)
  7. Cole a URL no formulário desejado
