Painel Administrativo

📅 Gerenciar Programação Horária

Configure um arquivo MP3 para cada hora do dia (00:00 às 23:00)

ℹ️ Instruções

  1. Faça upload dos arquivos MP3 no Cloudinary
  2. Copie a URL pública de cada arquivo
  3. Cole a URL no campo correspondente à hora desejada
  4. Marque "Ativo" para habilitar a reprodução naquela hora
  5. Use o botão "Testar" para verificar se o áudio está funcionando
  6. Clique em "Salvar" para aplicar as mudanças
⚠️ Segurança: Este painel usa a chave anônima do Supabase. Para produção, implemente RLS (Row Level Security) adequado no Supabase ou mova as operações de escrita para um backend seguro.
